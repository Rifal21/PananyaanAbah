# SundaAI - Asistén Pribadi Basa Sunda

Aplikasi wébsite chat AI modern anu dirancang khusus kanggo Basa Sunda, nganggo teknologi Google Gemini.

## Fitur

- **Basa Sunda Wungkul**: AI diprogram kanggo nyarios sareng ngartos Basa Sunda.
- **Desain Modern**: Antarbeungeut anu endah (Glassmorphism), responsif, sareng poék (Dark Mode).
- **Format Teks**: Ngarojong Markdown sareng Syntax Highlighting (mun nanya kode program).
- **Serverless**: Jalan langsung di browser anjeun tanpa backend nu ribet.

## Cara Ngawitan

1. **Buka Aplikasi**:
   Cukup buka file `index.html` dina browser (Chrome, Edge, Firefox, jsb).

   Atanapi anggo server lokal kanggo performa nu langkung saé:

   ```bash
   npx serve .
   # atanapi
   python3 -m http.server
   ```

2. **Masukkeun API Key**:
   - Nalika dibuka, anjeun bakal dipenta **Google Gemini API Key**.
   - Kengingkeun gratis di [Google AI Studio](https://aistudio.google.com/app/apikey).
   - API Key anjeun disimpen aman dina _Local Storage_ browser anjeun (henteu dikirim ka server urang).

## Teknologi

- HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- Google Gemini API (model `gemini-1.5-flash`)
- Font Awesome (Ikon)
- Google Fonts (Outfit & JetBrains Mono)
- Marked.js (Markdown parser)
- Highlight.js (Syntax Highlighting)
